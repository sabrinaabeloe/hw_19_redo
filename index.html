    Hi, I am Sabrina Abeloe and I am a Theatre Technician turned UX Designer. I thrive in fast-paced environments that require solid critical thinking and problem-solving skills.

    My Skills

    My skills include proficiency in Miro, Figma, Adobe Creative Suite, and Visual Studio Code. I use these skills to help make the UX/UI journey easier and more efficient for the user and the business. Figma is a tool used to help map out the User's journey when navigating the product and input the ideas for the interface. Miro is an organizational tool used to help keep the details from my user interviews in one place. I used the adobe creative suite products to design anything that I cannot do in figma and Visual Studio Code to execute the whole process into a product.
